ignite 2019, what’s new with containers and kubernetes on azure?

2019-11-04
#containers  #kubernetes  #helm  #azure  #azure-devops 

I won’t go through all the announcements made at the conference Microsoft Ignite 2019, you could find all of them all-in-one-place here: Book of News, Ignite 2019.

Here below, what I would like to highlight, are the updates and news around Docker and Kubernetes on Azure, but first have a look at this blog announcement: Accelerating cloud-native application development in the enterprise.

Azure Kubernetes Service (AKS) #

  • Release 2019-10-28
  • Authenticated IPs is GA
  • NodePools is GA
  • Autoscaler is GA
  • Availability Zones is GA
  • Standard Load Balancer is GA
  • Azure Application Gateway Ingress Controller (AGIC) is GA
  • Threat protections in Preview
  • Easier diagnostics and logging in Preview
  • Dev Spaces in Preview
  • Managed identities integration in Preview

Azure Container Registry (ACR) #

  • OCI artifact support is GA
  • UAE North region is GA
  • Scan container images for vulnerabilities with Qualys in Preview
  • Repository-scoped permissions in Preview
  • Teleport in Private Preview
  • Repository-scoped permissions in Preview
  • Diagnostic logs in Preview

Azure Monitor for containers #

  • Live performance metrics and live deployments is GA
  • Azure Monitor Prometheus integration is GA
  • Azure US Government region is GA
  • Containers for China regions, Grafana dashboard template, Agent - October 2019 updates
  • Support for containers on-premises and on Azure Stack in Preview
  • No instrumentation APM for Application on Kubernetes in Private Preview

Miscellaneous #

  • Azure Arc
  • Azure RedHat OpenShift hourly prices
  • Azure SQL Edge in Preview
  • Azure Spring Cloud in Preview
  • Azure API Management Self-hosted Gateway
  • Dapr Distributed Application Runtime (Dapr)
  • Improved CD capabilities and caching for Azure Pipelines
  • Secure software supply chain with Azure Pipelines artifact policies
  • Helm 3.0.0 is out!

That’s a lot of excitement, isn’t it!? Love it!

Here are few sessions you may want to watch on-demand:

  • Arc
  • OAM and Dapr
  • ASC
  • KEDA

Cheers!

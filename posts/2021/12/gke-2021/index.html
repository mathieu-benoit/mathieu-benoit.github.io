gke and anthos in 2021, a year of innovations

2021-12-31
#gcp  #kubernetes  #service-mesh 

Before starting 2022, let’s highlight what was announced and released in 2021 around GKE and Anthos.

GKE #

  • 1.17.14 to 1.21.6 went GA and 1.22.3 in Preview
  • Autopilot
  • Image Streaming
  • Multi-cluster services
  • Gateway controller
  • Backup
  • Cloud DNS
  • Cost Optimization Metrics
  • Internal TCP/UDP load balancer with Private Service Connect
  • Authenticating with Identity Service

Google Cloud Platform Podcast - GKE turns 6!

Anthos #

  • 1.5.3 to 1.9.2 went GA
  • Bare metal 1.10
  • VMware 1.10
  • GKE on AWS and Azure GA
  • VMs
  • Distributed Cloud

Cloud Run for Anthos (CRfA) #

  • 0.19 to 0.23 went GA
  • Integration with ASM
  • Knative applies to become a CNCF incubating project

Anthos Service Mesh (ASM) #

  • 1.6.14 to 1.12.0 went GA
  • Managed Anthos Service Mesh (managed control plane + data plane)
  • Distroless proxy image

Anthos Config Management (ACM) #

  • 1.6.1 to 1.10.0 went GA
  • Config Controller
  • Helm and Kustomize support

Misc #

  • GKE and containers - Top 10 blog posts of 2021
  • The past, present, and future of Kubernetes with Eric Brewer
  • A cloud built for developers — 2021 year in review
  • Google’s CNCF contributions
  • Anthos 101 learning series: All the videos in one place
  • Cloud Deploy
  • Managed Service for Prometheus
  • 2021 Accelerate State of DevOps report

That’s a wrap! More to come in 2022 for sure! Happy sailing! ;)

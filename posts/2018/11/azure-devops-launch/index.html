azure devops - community launch event in quebec city

2018-11-28
#azure  #azure-devops  #presentations 

Last night, we hosted in Le Camp the Azure DevOps Community Launch Event in Quebec city. It’s a world wide event to talk about the latest and greatest around Azure DevOps: https://www.azuredevopslaunch.com/

The agenda was:

  • Introduction to Azure DevOps by Mathieu Benoit, CSA at Microsoft and AzureQC Meetup organizer
  • Open Source projects with Azure DevOps by Hinault Donfack, Microsoft Dev Tech MVP and DotNetQC Meetup organizer
  • Mobile development with Xamarin and Azure DevOps by Hamida Rebai, Microsoft Dev Tech MVP
  • Docker and Kubernetes with Azure DevOps by Julien Maitrehenry, Microsoft Azure MVP and DockerQC Meetup organizer

We were 30 people sharing expertise and knowledge around Azure DevOps, really love this kind of session driven by the community eager to share and learn!
We got Subway lunch boxes and Azure DevOps t-shirts and stickers, thanks Microsoft and the local community!

To complete this, here are some resources you could leverage that I have written about Azure DevOps:

  • Tutorial: Using Azure DevOps to setup a CI/CD pipeline and deploy to Kubernetes
  • Docker, Kubernetes, DevOps and OSS at the Agile Tours Montreal and Quebec city
  • Azure DevOps to deploy your apps/services into a Kubernetes cluster
  • Helm charts repository with Azure Container Registry
  • Global DevOps Bootcamp 2018 - second edition

Cheers!
